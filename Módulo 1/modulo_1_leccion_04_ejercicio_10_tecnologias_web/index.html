Tecnologías web

Existen numerosas tecnologías web, algo que si bien es una ventaja en el diseño y desarrollo de sitio web, puede llegar a complicar la elección o la toma de decisión de cuál utilizar y para qué. Por eso es imprescindible explicar en detalle el proyecto que se quiere desarrollar, las funciones que necesitará el desarrollo web, y en base a eso, dejarse asesorar por los desarrolladores a la hora de determinar qué tecnologías web se usarán.
  • HTML
  • CSS
  • JavaScript
